# Gayatri Time Roadmap

This roadmap captures the next stages for Gayatri Time after the Android APK
release. The goal is to move from a working app toward a mature, reliable, and
spiritually useful daily companion.

## Version 1.1 - Polish

- Check the interface on different screen sizes.
- Test on Android 11 through Android 16.
- Verify battery saver behavior and long-running notification reliability.
- Fix small visual rough edges.

## Version 1.2 - Expansion

- Favorite cities.
- Multiple saved locations.
- Settings backup.
- Schedule export.
- Home screen widget.

## Version 2.0 - Spiritual Companion

- Daily Gayatri mantra.
- Short morning recommendations.
- Explanations for each sacred time window.
- Beautiful reminders before Brahma Muhurta.
- Practice journal.

At this stage, the app can grow beyond a calendar into a practical spiritual
assistant for daily practice.
